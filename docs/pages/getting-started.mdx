# Get started

To get started, navigate to the IIIF Manifest Editor (https://manifest-editor.digirati.services/)

![Landing page for Manifest Editor](https://digirati-workshops.pages.dev/open-manifest-editor.PNG)

If you have not accessed the Manifest Editor recently, you will be directed to the 'Examples' tab, where you can click on and open a number of example Manifests. These are presented as readonly previews.

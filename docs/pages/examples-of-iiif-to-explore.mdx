# IIIF content to Explore

## TU Delft

Given our location for the workshop today, where better to start than at TU Delft - you can navigate their IIIF collection from here: (https://heritage.tudelft.nl/iiif/collection.json )

Using Theseus from the viewer tools, you can navigate through the collection - simply open Theseus and paste in the Collection URL or use the link: 

(https://theseusviewer.org/?iiif-content=https://heritage.tudelft.nl/iiif/collection.json )
